# ⛯ VIGÍA · Control de Riesgo Laboral Zona Sur

## Despliegue en GitHub Pages

1. Crea un repositorio nuevo en tu cuenta de GitHub: `vigia`
2. Sube todos estos archivos al repositorio:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png` *(opcional, añade un icono 192×192px)*
   - `icon-512.png` *(opcional, añade un icono 512×512px)*
3. Ve a **Settings → Pages → Branch: main → / (root)** y guarda
4. La app estará disponible en: `https://lnovelm-ui.github.io/vigia/`

## Instalar en móvil (PWA)

- **Android Chrome:** entra en la URL, toca el menú (⋮) → *Añadir a pantalla de inicio*
- **iPhone Safari:** toca el icono de compartir → *Añadir a inicio*

## Base de datos

Conectada a Supabase (proyecto `lnovelm-ui's Project`).
Tablas creadas: `inspecciones_itss` y `litigiosidad`.

## Importar datos históricos

Desde la app, botón **⬆ Importar Excel** — acepta el Excel de Open Bravo
depurado directamente (detecta las columnas automáticamente).
